WebRTC: déployer du P2P dans le navigateur

WebRTC permet de créer des applications multimédia temps réel (voix, vidéo, chat, …) et de réaliser du transfert de données, le tout en mode P2P. Nous nous intéresserons à sa mise en oeuvre dans le navigateur et à l’infrastructure qui permet de la supporter.
 Stéphane Alnet

Svelte : Un concurrent de poids face à Angular-React-Vue ?

Svelte n'est pas une librairie, ni un framework mais un compilateur générant du VanillaJS à partir d'un fichier `.svelte`. Cette approche apporte plusieurs intérêts donc principalement : Meilleurs performances (le JS est optimal), Sucre syntaxique (des raccourcis permettent d'écrire moins de code et donc produire plus) et l'encapsulation CSS native.
 Cantin_Al
